TRMNL PLUGIN

RuneScape HiScores

A TRMNL Recipe, powered by a small Cloudflare Worker microservice, for RuneScape HiScores ranks.

Get RuneScape recipe Old School recipe coming soon

RuneScape

GET /api/rs3/summary?mode=player&name=Zezima
GET /api/rs3/summary?mode=ironman&name=Zezima
GET /api/rs3/summary?mode=hardcore_ironman&name=Zezima
GET /api/rs3/summary?mode=leagues&name=Zezima

Old School

GET /api/osrs/summary?mode=player&name=Zezima
GET /api/osrs/summary?mode=ironman&name=Zezima
GET /api/osrs/summary?mode=hardcore_ironman&name=Zezima
GET /api/osrs/summary?mode=ultimate_ironman&name=Zezima
GET /api/osrs/summary?mode=deadman&name=Zezima
GET /api/osrs/summary?mode=leagues&name=Zezima

No API key or authentication is required.

While I would like to support Group Ironman, neither RuneScape nor Old School provide an API for this data in a feasible way. I hope this changes in the future 😌.
